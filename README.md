# 🎬 Movie Explorer

A modern React Native (Expo) application to discover movies & TV shows, see where to watch them, and save favorites — powered by TMDB.

Search, explore trending titles, check streaming providers by region, and build your personal watchlist.

## Features

- Search Movies & TV Shows

Debounced search (fast & smooth UX)

Movie / TV filter chips

- Trending Today

Shows trending movies & TV when search is empty

- Detailed View

Overview, rating, release year

Streaming providers by region (Subscription / Rent / Buy)

Direct Open on TMDB link

- Favorites

Save & remove favorites

Persistent storage

- Region Selection

See availability by country (US, TR, DE, GB)

- Polished UI

Modern card layout

Haptic tab navigation

Dark-friendly theme



## Tech Stack

- React Native (Expo)

- Expo Router

- TypeScript

- TMDB API

- Local storage for favorites & settings
